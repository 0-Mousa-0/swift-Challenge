RoomaAR Resources - 3D Models
-----------------------------

This folder should contain the 3D furniture models used by the AR experience.

Required placeholder files (replace with your own .usdz assets):
- sofa.usdz
- table.usdz
- chair.usdz
- curtain.usdz
- carpet.usdz

To use your own models on iPad:
1. Create or download lightweight .usdz files for each furniture type.
2. Add them into this Resources folder inside the RoomaAR.swiftpm package.
3. Make sure the file names exactly match the ones listed above.
